## 3.1. 리스트와 튜플
* 튜플(tuple): 불변(ommutable) -> 항목에 할당하면 바꿀 수 없다. 
* 리스트(list): 변경 가능(mutable) -> 항목에 할당하고, 자유롭게 수정하거나 삭제할 수 있다.
